Features ‚ú®

  • Framework-agnostic: Works with React, Vue, Vanilla JavaScript, and more.
  • Searchable: Easily filter options based on search input.
  • Multi-select: Supports single and multiple selections.
  • Async select: Dynamically load options based on user search input.
  • UI flexibility: The UI is fully customizable; you control how the options and selected items are displayed.

API Reference üìö

createSelect()

The createSelect() function is the core of the library and handles the logic for filtering, selecting, and managing options.

Parameters:

  • options (Array): The list of options to choose from.
  • asyncLoadOptions (Function, optional): A function to load options asynchronously.
  • isMulti (Boolean, optional): Whether multiple options can be selected.

Returns:

  • options: The list of options filtered by the search term.
  • selected: The list of currently selected options.
  • search(term: string): A function to set the search term and filter options.
  • select(option: object): A function to select or deselect an option.
